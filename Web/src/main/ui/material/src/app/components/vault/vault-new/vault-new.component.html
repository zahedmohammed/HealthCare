Vault Details

Store App's username, passwords, url as secret's and safely inject it into projects Fxfile's 'baseUrl', 'username', 'password', & test-suite files. Vault helps you avoid sensitive environment details in version-control system.

e.g. usage password: '{{@Vault.FxLabs/MyPassword}}'

Where 'FxLabs/MyPassword' is the secret name.
