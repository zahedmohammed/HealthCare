Fx Gradle Plugin

plugins { id "io.fxlabs.job" version "0.2.10" }

     fx {

     jobId = "{{data}}"
     username = "{fx_username}"
     password = "{fx_password}"
     region = "{bot_region}"

}

To use fxlabs gradle plugin to invoke job include below script in build.gradle.

And replace username, password and region with your details from fxlabs and run the below command

gradle fx

which invokes the build on fxlabs
