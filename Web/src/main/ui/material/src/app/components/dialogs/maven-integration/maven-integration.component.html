Maven Invoke Plugin

Add the plugin to pom.xml in plugin sections

<plugin>
   <groupId> io.fxlabs </groupId>
   <artifactId> fxt-job-maven-plugin </artifactId>
   <version> 2.0.0 </version>
   <configuration>
       <username> "{fxusername}" </username>
       <password> "{fxpassword}" </password>
       <jobId> "{{data}}" </jobId>
   </configuration>
</plugin>

For latest plugin version go to https://search.maven.org/artifact/io.fxlabs/fxt-job-maven-plugin

After adding plugin to your pom. Replace fxusername and fxpassword with your details from fxlabs

Then run below command in project folder

mvn clean fxlabs:job
