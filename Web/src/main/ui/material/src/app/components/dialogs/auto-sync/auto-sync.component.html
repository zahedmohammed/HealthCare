AutoCode (Developer Bot - Automatic test coverage)

1. Auto-generates test-suites based on the project's AutoCode configuration.

2. AutoCode never overrides an existing test-suite file it only creates new files.

3. Sync's new or updated files from the Git repository.

4. Delete (Optional)

      Select All {{category}}

    Previously auto-generated test-suites of selected categories files will be deleted.
    Note: Files which are not created by AutoCode are not deleted.
5. Create (Optional)
    Select this option if you made changes to the AutoCode's configuration and would like to generate test suites.
    Note: AutoCode will generate test-suite files based on the AutoCode's configuration settings.
